# MOH Coop Trilogy - release pipeline

Build + packaging tooling for the MOH Coop Trilogy mod (HZM Extended, MOHAA War Chest on OpenMOHAA).

- build.ps1 - packs the mod tree into the coop pk3 and deploys to the local dev install
- installer/ - Inno Setup installer sources (hzm_coop.iss), packaging script, problem reporter

Source repos: [engine](https://github.com/MOHCoopTrilogy/openmohaa) | [mod](https://github.com/MOHCoopTrilogy/hzm-mohaa-coop-mod)

Game releases and the auto-update manifest are published on this repo's Releases page.
